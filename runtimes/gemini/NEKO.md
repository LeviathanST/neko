## Memory

You have neural memory available via MCP tools. Use it:
- `nmem_recall` — retrieve past context before starting work
- `nmem_remember` — save decisions, insights, patterns after completing tasks
- `nmem_explain` — trace connections between concepts
- `nmem_hypothesize` / `nmem_predict` / `nmem_verify` — cognitive reasoning

**Rule:** Each session starts fresh. Without explicit saves, ALL discoveries are lost forever.
